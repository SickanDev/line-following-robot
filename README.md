# line-following-robot

Basically a robot that's able to follow a line.

## Components connected to board

- KY-033 Tracking sensor module (2x)
  - 2DP
- SparkFun Motor Driver - Dual TB6612FNG (1x)
  - 7DP

## Ports

- Digital
  - In use = 10
  - Available = 1
- Analog
  - In use = 0
  - Available = 6
